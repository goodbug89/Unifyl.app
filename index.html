Features

Everything you need.
Nothing you don't.

Power features that rival Total Commander, wrapped in a native macOS experience that feels right at home.

⚙

Dual-Pane Power

Side-by-side file browsing with tabs, workspaces, and keyboard-first navigation. Copy, move, and compare files across panes with function keys.

🔍

Advanced Search

Multi-criteria search with regex, size, date, content matching, and boolean logic. Results feed directly into Smart Folders for batch operations.

📦

Archive Virtual Folders

Browse ZIP, 7z, TAR, GZ, RAR archives as regular folders. Copy, move, rename, and delete files inside archives seamlessly.

10 Built-in Viewers

Text, hex, image, PDF, video, audio, web, Markdown, code with syntax highlighting, and media info. Each in its own resizable window.

Themes & Icon Packs

12 built-in themes with full customization. SVG icon packs with 200+ extension mappings. Make it yours.

Multi Rename

Batch rename with regex, numbering, EXIF data, search-replace, and AI suggestions. Preview before applying, undo after.
